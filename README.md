# HackerNews Trends Project

## Description
This project fetches top trending stories from HackerNews API, categorizes them into different categories, and stores them in a JSON file.

## Features
- Fetches top 800 stories
- Categorizes into 5 categories
- Limits 25 stories per category
- Ensures 100+ stories using fallback logic
- Saves output into JSON file

## Technologies Used
- Python
- Requests library
- HackerNews API
